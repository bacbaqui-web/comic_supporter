마감이즈커밍 개인정보처리방침

시행일: 2026년 6월 21일

마감이즈커밍은 웹툰 작가의 작업을 돕기 위해 일정, 메모, 북마크, 노동요, CLIP 미리보기 관리 기능을 제공하는 웹앱입니다.

마감이즈커밍은 별도의 자체 서버에 사용자의 작업 데이터나 개인 데이터를 저장하지 않습니다. 앱 데이터는 사용자의 브라우저와 사용자가 로그인한 Google Drive에 저장됩니다.

1. 처리하는 정보

마감이즈커밍은 서비스 작동에 필요한 범위에서 다음 정보를 처리할 수 있습니다.

  • Google 로그인 시 제공되는 기본 프로필 정보: 이름, 이메일, 프로필 이미지
  • 사용자가 앱에 입력하거나 저장한 일정, 메모, 북마크, 노동요 목록, CLIP 미리보기 데이터
  • 사용자가 업로드하거나 저장한 이미지, 북마크 미리보기 이미지, CLIP 미리보기 이미지
  • 자동 로그인 복구를 위한 브라우저 로컬 저장소 값

2. Google Drive 권한 사용

마감이즈커밍은 Google Drive API의 drive.file 권한을 사용합니다. 이 권한은 앱이 사용자의 Google Drive 안에 생성하거나 사용자가 앱을 통해 선택/저장한 파일을 관리하기 위해 사용됩니다.

  • 사용자 Google Drive 안에 앱 데이터 폴더 생성
  • 달력, 메모, 북마크, 노동요, CLIP 미리보기 데이터를 JSON 또는 텍스트 파일로 저장
  • 북마크 이미지와 CLIP 미리보기 이미지를 사용자의 Google Drive에 저장
  • 같은 Google 계정으로 접속한 다른 기기에서 앱 데이터를 불러오기

3. 브라우저 저장소와 자동 로그인

마감이즈커밍은 로그인 성공 여부를 브라우저 로컬 저장소에 기록하여 다음 접속 때 Google 세션이 유지되어 있으면 로그인을 자동으로 복구할 수 있습니다. 로그아웃하면 이 값은 삭제됩니다.

Google 액세스 토큰은 브라우저 메모리에서만 사용되며, 영구 저장용 refresh token을 앱 자체에 저장하지 않습니다.

4. 외부 서비스

마감이즈커밍은 기능 제공을 위해 다음 외부 서비스를 사용할 수 있습니다.

  • Google OAuth, Google Drive API, Google Fonts
  • YouTube Data API, YouTube iframe player, YouTube 썸네일 및 oEmbed
  • 동영상 제목 및 재생목록 정보를 보완하기 위한 공개 메타데이터 조회 서비스

외부 서비스 이용 과정에서 사용자의 브라우저가 해당 서비스에 요청을 보낼 수 있으며, 각 서비스의 개인정보처리방침과 이용약관이 적용될 수 있습니다.

5. 데이터 저장 위치

앱 데이터는 사용자의 브라우저와 사용자의 Google Drive에 저장됩니다. 서비스 운영자는 사용자의 Google Drive 파일을 별도 서버에 복사하거나 저장하지 않습니다.

6. 데이터 공유

마감이즈커밍은 사용자의 개인 데이터나 작업 데이터를 판매하지 않으며, 운영자가 별도 서버를 통해 제3자에게 제공하지 않습니다.

7. 데이터 삭제와 권한 철회

사용자는 Google Drive에서 앱 데이터 폴더 또는 그 안의 파일을 직접 삭제할 수 있습니다. 해당 파일을 삭제하면 앱에서 불러오는 데이터도 삭제됩니다.

사용자는 Google 계정 설정에서 언제든 마감이즈커밍의 Google Drive 접근 권한을 철회할 수 있습니다. 브라우저의 사이트 데이터 삭제 기능을 사용하면 자동 로그인 복구 값 등 로컬 저장 데이터도 삭제할 수 있습니다.

8. 보안

마감이즈커밍은 Google OAuth와 Google Drive API를 통해 사용자의 권한을 요청합니다. 다만 브라우저, 네트워크, Google API 상태, 사용자의 계정 설정에 따라 로그인, 저장, 동기화가 실패할 수 있습니다.
